# README #

### Prerequisites ###
* Nodejs (>12)
* nvm
* yarn

### How do I get set up? ###
* Clone git repository using `git clone`
* Run `yarn install` in terminal to get all dependencies

### How do I run the tests? ###
* To run:
  - All tests:
  ```bash
  $ yarn wdio
  ```
  - Time API suite:
  ```bash
  $ yarn time-tests
  ```
  - World API test: 
  ```bash
  $ yarn world-tests
  ```
  - Feeling Lucky test:
  ```bash
  $ yarn google-tests
  ```
* Screenshots will be available on `/media/`
